The following three audio clips included in this page are proof-of-concept examples of the installation's sound. These clips were made as an exploration on what auditory mood we want to achieve for this installation. The sound portion of the installation will be generated using Max, due to its ability for incorporating sensory input for changing the output of the audio.


Audio snippets by Sherman Fong


This audio snippet aims for a serene and mellow mood. This clip uses several percussion instruments and synths in order to create something etheral, like a symphony of light in audio form.


This snippet aims to replicate a somewhat dark atmosphere. This track can be described as a sombre take on new age music.


This snippet is an attempt at creating a dark ambient soundscape. It aims to be unsettling to listen to.
